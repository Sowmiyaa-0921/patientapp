🏥 PatientApp – App Description

PatientApp is a simple and user-friendly Android application designed to manage patient information and communicate with doctors easily. The app allows users to enter patient details, select illness type, schedule appointments, and quickly contact the doctor via call, SMS, or email.

✨ Features

📝Enter patient details (Name, Age, Gender)
🦠 Select illness from dropdown list
📅 Choose appointment date using Date Picker
📞 Call doctor directly from the app
💬 Send SMS to doctor
📧 Send email for appointment request
🎨 Clean and easy-to-use interface
🎯 Purpose of the App

The purpose of PatientApp is to:

Simplify appointment booking
Provide quick communication with doctors
Organize patient information in one place
Demonstrate Android UI components and Intents

🛠 Technologies Used

Java
XML
Android Studio
Intent (Call, SMS, Email)
Spinner
RadioButton
DatePickerDialog

📌 Target Users

Patients
Clinics
Medical students (for learning Android development)
